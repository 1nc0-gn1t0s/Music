MUSIC KEEPER




Техническое задание:


Сервис для хранения музыки с автоматическим подбором обложки для трека при помощи генерации картинок по тексту с использованием Chat GPT. Пользователь может сохранять треки, для чего ему нужно будет загрузить файл с компьютера и написать различные параметры, такие как название, текст и исполнитель. После этого по тексту песни (или по ее названию, если текста нет) будет генерироваться картинка-обложка и цвет, соответствующий «настроению» песни. 




Ресурсы:


1. Flask
2. OpenAI GPT API
