A service for storing music with automatic selection of cover art for a track by generating images from text using Chat GPT. The user can save tracks, for which he will need to download the file from the computer and write various parameters such as title, lyrics and artist. After this, a cover image will be generated based on the lyrics of the song (or its title, if there are no lyrics).
